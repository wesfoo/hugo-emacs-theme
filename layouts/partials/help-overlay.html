Keyboard Shortcuts

{{/* Emacs keybindings - shown when Evil mode is off */}}

Navigation (List)

n / ↓ Next article
p / ↑ Previous article
RET / o Open article
q Go back to list
< Beginning of list
> End of list

Windows

C-x 3 Split horizontal (side by side)
C-x 2 Split vertical (stacked)
C-x 0 Close current window
C-x 1 Close other windows
C-x o / Tab Switch to other window
C-x b Switch to list buffer

Scrolling (Content)

n / p Scroll down/up (split) or next/prev article (single)
↑ / ↓ Scroll up/down
PgUp / PgDn Page up/down
SPC / S-SPC Page down/up
C-v / M-v Page down/up (Emacs style)

Evil Mode (Vim keybindings)

Navigation (List)

j / ↓ Next article
k / ↑ Previous article
RET / l Open article
h / q Go back to list
gg / H Beginning of list
G / L End of list

Windows

Ctrl-w v Split vertical (side by side)
Ctrl-w s Split horizontal (stacked)
Ctrl-w c Close current window
Ctrl-w o Close other windows
Ctrl-w w / Tab Switch to other window
Ctrl-w h/j/k/l Focus window by direction

Scrolling (Content)

j / k Scroll down/up
n / N Next/previous article
Ctrl-d / Ctrl-u Half page down/up
Ctrl-f / Ctrl-b Full page down/up
gg / G Beginning/end of buffer
h / q Go back to list
