Keyboard Shortcuts

Navigation (Dired-style)

n / C-n / ↓ Next article (with preview)
p / C-p / ↑ Previous article (with preview)
RET / o / SPC Open article and focus content
q Quit / Go back
< / M-< Go to beginning of list
> / M-> Go to end of list

Windows

C-x o / Tab Switch to other window
C-x 1 Focus article list
C-x 2 Focus article content

Scrolling (in content window)

C-v / SPC Scroll down (Page Down)
M-v / S-SPC Scroll up (Page Up)
n / p Scroll line by line

View

t Toggle Modus Vivendi/Operandi theme
+ Increase font size
- Decrease font size
